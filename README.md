# 🦖 Raptor Solutions: Secure Admin Portal & RAG Assistant

Un sistema de gestión administrativa Fullstack con arquitectura segura (RBAC) y un asistente de Inteligencia Artificial Generativa (RAG) integrado, diseñado para la gestión de infraestructura IT y seguridad.

## 🚀 Arquitectura del Proyecto

Este proyecto combina prácticas sólidas de ciberseguridad con AI Programming moderno:
- **Frontend:** React + TypeScript (en desarrollo)
- **Backend:** Python + FastAPI (en desarrollo)
- **Base de Datos:** PostgreSQL
- **AI Core:** Arquitectura RAG para consulta de documentos técnicos con control de acceso.

## 🛡️ Características Principales
* **Autenticación Segura:** JWT y encriptación de contraseñas.
* **Control de Acceso Basado en Roles (RBAC):** Restricción de endpoints según el perfil del usuario.
* **Asistente IA Contextual:** Respuestas generadas a partir de documentos internos, respetando los permisos de lectura del usuario.

## ⚙️ Estructura del Repositorio
El proyecto está dividido en dos microservicios principales: `frontend` y `backend`.
